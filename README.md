# LukSuwa Final Project

LukSuwa is a final year university project focused on building an AI-powered e-health and medicine management platform. The system combines mobile healthcare support, telemedicine, pharmacy services, AI-based medicine assistance, and symptom prediction into one integrated solution.

## Project Overview

The main goal of LukSuwa is to improve access to healthcare-related services through a mobile application. The system is designed for multiple types of users, including patients, doctors, pharmacies, and administrators. It also integrates AI-powered features to support healthcare interactions and improve the overall user experience.

## Main Features

- User authentication and role-based access
- Patient, doctor, pharmacy, and admin dashboards
- Telemedicine and video calling support
- Prescription-related features
- Pharmacy inventory management
- AI-based medicine assistant
- Symptom prediction module
- Notifications and profile management
- CSV upload support for pharmacy inventory
- Google Maps integration for nearby pharmacy and location features
